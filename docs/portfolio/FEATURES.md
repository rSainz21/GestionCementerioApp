# Funcionalidades (selección)

## Web (gestión)

- **Dashboard** con KPIs de ocupación y accesos rápidos.
- **Gestión de datos base**: cementerios, zonas, bloques, unidades, concesiones.
- **Búsqueda** por DNI / nombre / expediente.
- **Nuevo caso** guiado: concesión + asignación de unidad + datos asociados.

## App de campo

- **Mapa y navegación** por zonas/bloques/unidades.
- **Ficha de sepultura**: datos clave, ubicación, notas, documentos/fotos, acciones.
- **Flujos operativos**: inhumación, exhumación/traslado, registro de movimientos.
- **Modo offline**: cola de cambios y sincronización posterior.

## API

- Endpoints de **listado/búsqueda** (concesiones, terceros, difuntos, sepulturas).
- Endpoints de **ficha**, **actualización**, **workflow** y **adjuntos**.
- **Permisos** por endpoint (ver/crear/editar/admin).
