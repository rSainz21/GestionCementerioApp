## Contribuir

Este repo está orientado a **portfolio**. Aun así, si quieres proponer mejoras:

1. Abre un issue con contexto y capturas (sin datos personales).
2. Crea un PR pequeño, con descripción y plan de pruebas.
3. No incluyas:
   - credenciales reales,
   - dumps de base de datos,
   - ficheros subidos (storage),
   - `node_modules/` ni artefactos de build.

### Estilo de commits

- Mensajes cortos y orientados a intención: `feat:`, `fix:`, `docs:`, `refactor:`.
